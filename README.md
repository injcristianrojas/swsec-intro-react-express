# React-Express Software Security

Based on https://github.com/Joao-Henrique/React_Express_App_Medium_Tutorial

# Launch

## Launch API

```shell
cd api
npm install
npm start
```

## Launch SPA (client)

```shell
cd spa
npm install
npm start
```
